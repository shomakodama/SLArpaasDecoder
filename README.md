# SLArpaas Decoder

## Overview

This code is written and maintained by **Shoma Kodama** (The University of Tokyo; `shoma@hep.phys.s.u-tokyo.ac.jp`).

## Basic Explanation

The code is written in **C++** used with **ROOT**.
